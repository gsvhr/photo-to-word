# Фототаблица

Приложение для создания таблиц с фотографиями в Microsoft Word.

## Возможности

- Загрузка изображений различных форматов (jpg, png, bmp, gif, tiff, webp, avif)
- Удобный просмотр миниатюр с возможностью выбора
- Гибкие настройки генерации:
  - Одна или две колонки
  - Книжная или альбомная ориентация
  - Качество JPEG (50-100%)
- Сохранение в буфер обмена или на диск
- Прогресс генерации с возможностью отмены
- Горячие клавиши:
  - Ctrl+F - загрузить фото
  - Ctrl+Del - очистить все
  - Del - удалить выбранные
  - Ctrl+A - выделить все
